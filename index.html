Reprovider

A service injector designed for React and Redux

Read the docs

Easy to Use

Reprovider is a centralized provider that allows you inject services quickly into a react component using a consumer.

Dependecy Injection

Reprovider use the Dependency Injection principle for insert the services into components without needed instantiate the classes.

Redux Integration

Reprovider can inject services into reducers and thunks using Buildux
